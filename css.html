As principais ESTRUTURAS do CSS são:

  • FONT-FAMILY define a família de fontes a ser usada para o texto.

  • FONT-SYZE define o tamanho da fonte.

  • FONT-WEIGHT define a espessura da fonte, como normal, negrito, etc.

  • FONT-STYLE define o estilo da fonte, como normal, itálico, obliqua, etc.

  • COLOR define a cor do texto.

  • TEXT-ALIGN alinha o texto horizontalmente dentro do elemento, como left, center, right, justify.

  • TEXT-DECORATION adiciona decoração ao texto, como sublinhado, tachado, etc.

  • TEXT-TRANSFORM controla a capitalização do texto, como maiúsculas, minúsculas, capitalização de palavras, etc.

  • WIDTH E HEIGHT define a largura e a altura da imagem.

  • MAX-WIDTH E MAX-HEIGHT define a largura e a altura máximas que a imagem pode ter, impedindo que ela ultrapasse esses limites.

  • MARGIN E PADDING controla a margem e o preenchimento ao redor da imagem.

  • BORDER adiciona uma borda à imagem.

  • BORDER-RADIUS define o raio dos cantos da imagem, criando bordas arredondadas.

  • BOX-SHADOW adiciona uma sombra à imagem.

  • OPACITY define a transparência da imagem.

  • OBJECT-POSITION define a posição inicial da imagem dentro do contêiner.
