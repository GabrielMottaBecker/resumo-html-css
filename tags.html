Principais tags do HMTL

Página Inicial

As principais TAGS utilizadas no HTML são:

  • DOCTYPE define o tipo de documento e a versão do HTML sendo usada.

  • HTML define o início e o fim do documento HTML.

  • HEAD contém informações sobre o documento, como título, metadados e links para estilos e scripts

  • TITLE define o título da página, exibido na barra de título do navegador ou na guia

  • BODY contém todo o conteúdo visível da página, como texto, imagens, links e outros elementos.

  • H1 TO H6 define os cabeçalhos, do maior (h1) ao menor (h6), indicando diferentes níveis de importância.

  • P define um parágrafo de texto.

  • A cria um hyperlink, permitindo que os usuários naveguem para outras páginas da web.

  • IMG insere uma imagem na página.

  • UL define uma lista não ordenada.

  • OL define uma lista ordenada.

  • LI define um item de lista em uma lista ordenada (ol) ou não ordenada (ul).

  • DIV cria uma divisão ou seção genérica na página, muitas vezes usada para agrupar e estilizar outros elementos.

  • MAIN define o conteúdo principal da página. Geralmente, é usado para envolver o conteúdo central da página, excluindo cabeçalhos, rodapés e barras laterais.

  • SECTION define uma seção genérica em um documento HTML, frequentemente usada para agrupar conteúdo relacionado. Pode ser útil para dividir o conteúdo de uma página em seções distintas.

  • HEADER define o cabeçalho de um documento ou de uma seção. Normalmente, é usado para incluir elementos como logotipos, menus de navegação e outras informações introdutórias.

  • FOOTER define o rodapé de um documento ou de uma seção. Geralmente, é usado para incluir informações de contato, links de copyright e outros detalhes relevantes.

  • NAV define uma seção de navegação. É comumente usado para agrupar links de navegação, como menus de navegação principal ou secundário.
